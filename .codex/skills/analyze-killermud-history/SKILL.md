# Analyze KillerMUD History

Read `lore/` only. Query a bounded set with `.codex/skills/query-killermud-lore/scripts/query_lore.py`, normally using domains `history`, `politics`, and `legends` plus `--include-related`.

Build chronology only from explicit or relative temporal evidence. Keep unknown dates unknown. Separate accepted history from legend, belief, rumor, and disputed accounts. Explain causes and consequences only when supported by claims or clearly label the inference. Include relevant record IDs and source references.

Do not add records, resolve canon, or generate area content.
